# FS25_OperatingCosts
Realistic Operating Costs adds regular monthly operating costs to Farming Simulator 25. The mod takes vehicles, implements, buildings, silos and stored goods into account, creating a more realistic long-term cost structure for your farm.

<ins>**Building Maintenance**</ins>

Owned buildings and placeables receive a target maintenance value. Existing maintenance costs from the base game or other mods are taken into account, so only a possible difference is added.

<ins>**Stored Goods Insurance**</ins>

Stored goods can be included through stored goods insurance. The calculation is based on the estimated value of the stored products.

<ins>**Ingame Overview**</ins>

The operating costs overview can be opened by default with:

*CTRL + ALT + B*

The key binding can be changed in the game’s control settings. The overview shows saved costs by category and time period. Monthly, yearly and total views are supported.

<ins>**Configuration**</ins>

When the savegame is saved, the mod creates the following file:

operatingCostsConfig.xml

The monthly percentages for each economic difficulty can be adjusted there. Changes are applied after reloading the savegame.

<ins>**Installation**</ins>

Download FS25_OperatingCosts.zip.
Place the ZIP file unchanged into your Farming Simulator 25 mods folder.
Activate the mod when starting your savegame.

The ZIP file should not be unpacked.

<ins>**Multiplayer**</ins>

In multiplayer, the calculation runs on the server or host. This helps prevent duplicate client-side bookings.

<ins>**Known Limitations**</ins>

Very specific storage systems from maps or building mods may not always be fully detected.
Static map buildings without usable price data cannot always be valued realistically.
The default values are intended as a gameplay-oriented baseline and can be adjusted per savegame.
